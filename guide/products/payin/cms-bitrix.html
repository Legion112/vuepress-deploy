# Настройка приёма платежей

Чтобы использовать модуль, нужно подключить магазин к VR_Cube, получить доступ в Личный кабинет и там получить следующий набор параметров:

  • UUID договора - идентификатор договора;
  • ID продукта - идентификатор продукта;
  • Секретное слово - секретное слово продукта;

  1. В панели управления сайтом выберите Магазин — Настройки — Платежные системы. Выберите платёжную систему VR_Cube.

  2. Нажмите Добавить платёжную систему.

  3. Заполните форму Параметры платежной системы данными из таблицы:

[datatables autoWidth=true paging=false searching=false ordering=false info=false]

Параметр Описание
Обработчик VR_CUBE (VRcube)
Заголовок VR_CUBE
Название Оплата банковской картой
Активность Поставьте галочку
Тип оплаты Выберите из списка Эквайринговая операция

[/datatables]

Остальные поля оставьте, как есть.

4.Ниже заполните блок По умолчанию для всех типов плательщиков данными из таблицы:

[datatables autoWidth=true paging=false searching=false ordering=false info=false]
