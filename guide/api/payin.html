# Эквайринг

Пролистайте ниже для образцов кода, примеров запросов и ответов. Выберете образец кода во вкладке на удобном Вам языке

VR_CUBE предоставляет услугу «Платежный шлюз» (Payment Gate) для обработки карточных данных держателей банковских карт с помощью Банков-эквайеров (Acquiring Banks).

Base URLs: http://gateway.dev.acqp.ipsp.dev/front_web
